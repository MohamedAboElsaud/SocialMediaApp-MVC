# Installation
Clone the repository
```sh
$ git clone https://github.com/mohamedaboelsaud/FoodiQ.git
$ cd FoodiQ
```

Open the file `FoodiQ.xcodeproj` using Xcode 
Click on the play button at the top left corner to build and run the project
